# 🏞️ Local Turístico

Projeto da **Parte 1** do curso **Full-Stack**, utilizando apenas **HTML e CSS**.  
O objetivo é praticar estruturação de conteúdo e aplicação de estilos em uma página sobre um local turístico (**BUSAN**).
