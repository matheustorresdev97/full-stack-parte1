# 🎓 Formulário de Matrícula

Projeto da **Parte 1** do curso **Full-Stack**, utilizando apenas **HTML e CSS**.  
O objetivo é praticar a criação de **formulários complexos**, simulando uma ficha de matrícula para escola.
