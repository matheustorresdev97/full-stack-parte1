# 🛒 LP de Produto - Zingen

Projeto da **Parte 1** do curso **Full-Stack**, utilizando apenas **HTML e CSS**.  
O objetivo é criar uma **Landing Page de produto** e aplicar **responsividade** para diferentes tamanhos de tela.
